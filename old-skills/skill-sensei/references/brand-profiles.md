# Brand Profiles

## Pirog

- Prefix: `piro-`
- Use for: personal workflows, taste, heuristics, and Mike-specific defaults
- Primary color: `#00c88a`
- Secondary color: `#1f2937`
- Watermark asset: `assets/pirog-watermark.png`

## Tanaab

- Prefix: `tanaab-`
- Use for: shared company workflows, process, policy, and team-facing knowledge
- Primary color: `#ffffff`
- Secondary color: `#0f172a`
- Watermark asset: `assets/tanaab-watermark.svg`
- When searching for a new Tanaab base icon, prefer lighter artwork so it stays distinct from the dark Tanaab watermark treatment.

## Shared Rules

- Human-facing names do not include the brand prefix.
- Icons should be visually consistent across both brands.
- Brand profile watermark entries should match exact filenames in assets/, not logical placeholder names.
- Watermarks should live in the lower-right corner at a fixed size and padding.
- If watermark assets are not available yet, generate a branded fallback icon and note the missing overlay as follow-up work.
